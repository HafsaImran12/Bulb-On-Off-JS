# 💡 Bulb On/Off Project

A simple beginner-friendly JavaScript project that demonstrates how to turn a bulb ON and OFF using HTML, CSS, and JavaScript. This project is perfect for learning basic DOM manipulation and event handling.

---

## 📸 Preview
![Bulb Preview](https://hafsaimran12.github.io/Bulb-On-Off-JS/)

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Features
- 💡 Turn bulb ON with a click
- 🔌 Turn bulb OFF with a click
- 🎨 Simple and clean UI
- ⚡ Beginner-friendly JavaScript logic
- 🔄 Instant DOM manipulation
